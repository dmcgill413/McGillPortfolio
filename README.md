# Personal Portfolio Website

This project is a multi-page personal portfolio built using semantic HTML, responsive CSS, accessibility best practices, and professional GitHub workflow. It showcases my skills, coursework, and future project goals as I continue learning web development.

## Pages Included
- index.html — Home / Introduction
- about.html — About Me
- projects.html — Portfolio & Work Samples
- contact.html — Contact & Social Links

## Features
- Mobile-first responsive design
- Flexbox navigation
- CSS Grid page layout
- Accessible alt text and descriptive link labels
- Consistent multi-page navigation
- Clean commit practices using GitHub Desktop
